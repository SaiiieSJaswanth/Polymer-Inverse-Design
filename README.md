# Polymer Electrolyte Dataset (Material Informatics)

This project contains a dataset of 100 polymer electrolyte systems used in batteries.

## Features
- Molecular Weight
- Ion Size
- Decomposition Temperature
- Ion Transference Number
- Humidity
- Degree of Polymerization
- Anion Radius
- Phase Transition Temperature
- Band Gap
- Expansion Coefficient
- Crosslink Density
- Processing Time

## Description
The dataset is synthetically generated using physically meaningful ranges and cleaned using data imputation techniques.
